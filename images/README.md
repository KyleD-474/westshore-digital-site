# Homepage images

Place your photos here and name them as below so the site shows them instead of the placeholders.

| File             | Where it appears | Suggested content |
| ---------------- | ----------------- | ----------------- |
| **hero.jpg**     | Top of page, under the headline | You, your workspace, or a laptop. Builds “real person, real business.” |
| **services.jpg** | Services section — first card (Computer Performance Upgrades) | Laptop/desk, speed upgrades, or you at a computer. |
| **services2.jpg**| Services section — second card (New Computer Setup & Data Transfer) | New computer setup, file transfer, or cables/desk. |
| **services3.jpg**| Services section — third card (In-Home Tech Help) | In-home visit, helping with device, or friendly tech support. |
| **headshot.jpg** | About Me section | A clear, friendly headshot. Builds trust — “this is who’s coming to your home.” |

**Tips**
- **Hero:** Landscape works best (e.g. 900×400px or similar). Clear, well-lit, not cluttered.
- **Services (all three):** One image per service card on desktop; on mobile only the first image shows. Same style across the three works well.
- **Headshot:** Face the camera, good light, neutral or approachable expression. Square crop (e.g. 400×400px) — the site displays it as a circle.
- **Format:** JPG is fine; PNG if you need transparency. Keep file size reasonable (e.g. under 300KB each) so the page stays fast for older devices.

Until you add the real photos, the page uses the SVG placeholders so the layout still works. In `index.html`, change the `src` from `.svg` to `.jpg` when you add each file.
